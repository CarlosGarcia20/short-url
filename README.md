## 📌 Descripción General
Aplicación Full Stack para el acortamiento de enlaces diseñada con una arquitectura desacoplada. Este sistema permite la generación dinámica de URLs personalizadas y la redirección de tráfico en tiempo real, garantizando alto rendimiento, persistencia de datos en la nube y protección contra abusos en las peticiones.

## ✨ Características Principales
* **Gestión Completa (CRUD):** Creación, lectura, actualización atómica y eliminación de enlaces directamente desde la interfaz.
* **Redirección Dinámica:** Resolución instantánea de URLs cortas hacia sus destinos originales.
* **Seguridad (Rate Limiting):** Límite de peticiones configurado en la API para prevenir saturación y asegurar la estabilidad del backend.
* **Persistencia NoSQL:** Modelado y almacenamiento de datos eficiente gestionado completamente en la nube.

## 🛠️ Stack Tecnológico
* **Frontend:** React, JavaScript, Vercel (CI/CD)
* **Backend:** Node.js, Express, Render (Hosting)
* **Base de Datos:** MongoDB Atlas
* **Documentación:** Mintlify
